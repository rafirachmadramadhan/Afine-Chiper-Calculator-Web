Welcome to KriptoProject.

This is a web for decryption/encryption with Affine cipher, an encryption function with addition and multiplication that encodes a letter into another letter with the value
(mP + b) mod 26 where m and b are the keys.
likewise a decryption function with
(m^-1)(C - b) mod 26.
